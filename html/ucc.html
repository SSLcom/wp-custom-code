UC Certificate (aka SANs, Exchange SSL, multidomain SSL) Summary

  • CAN secure up to 2000 entries (first 3 non-wildcard are free, additional charge per entry after the 3rd entry)
  • CAN use certificate on unlimited multiple servers concurrently
  • CAN change domains at anytime or specify them later without fees or costs
  • An entry CAN be a Fully Qualified Domain Name (FQDN)
  • An entry CAN be a single worded server name (but the certificate will not be effective past November 2015 per CAB Forum Reqs.)
  • An entry CAN represent a wildcard domain name (i.e. *.domain.com, but NOT multiple levels like *.*.domain.com)
  • CAN use on unlimited ip addresses with multiple, concurrent private keys (great for hosting and virtual hosts)
